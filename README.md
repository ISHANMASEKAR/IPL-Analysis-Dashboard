IPL Analysis Dashboard (2008–2025)

An interactive Power BI dashboard designed to analyze Indian Premier League (IPL) data from 2008–2025. This project transforms raw cricket data into meaningful visual insights through KPIs, charts, filters, and player/team performance analysis.

Project Overview

The dashboard provides a complete analysis of IPL seasons using historical data. It helps users explore trends related to teams, players, batting performance, bowling performance, toss decisions, and season-wise match statistics.

The goal of this project is to demonstrate data cleaning, transformation, data modeling, DAX calculations, and dashboard design skills using Power BI.


Dataset

Source:
https://www.kaggle.com/datasets/chaitu20/ipl-dataset2008-2025

- Match details
- Team information
- Ball-by-ball records
- Toss data
- Batter statistics
- Bowler statistics

- Match outcomes
